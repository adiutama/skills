# Summary format

Print before posting, then proceed immediately without asking for confirmation:

```
Session: <session_path>
Recommended stance: <from session file>

Unposted findings:
  C1 — <title>  path/to/file:LINE  [critical]
  W1 — <title>  path/to/file:LINE  [warning]
  N1 — <title>  path/to/file:LINE  [nit]

Posting as: <EVENT> | Personal message: <message or none>
```

If there are no unposted findings (`Posted: ❌`), say so and stop.
